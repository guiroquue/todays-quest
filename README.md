# Today's Quest

Turn self-imvement into a game.

Today's Quest is a gamified habit and self-improvement app for iPhone. Complete daily quests, earn experience, unlock badges, level up, and build a record of your personal growth.

## Features

- Daily quests
- Experience and leveling system
- Unlockable badges
- Progress tracking
- Memories and journaling
- Home Screen and Lock Screen widgets
- Local-first design
- No accounts required

## Privacy

Today's Quest does not collect personal data.

All progress, memories, badges, and settings are stored locally on the user's device.

See the Privacy Policy for more information.
